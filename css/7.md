# ***<u>CSS动画</u>***

```html
<!doctype html>
<html lang="ch">
<head>
    <meta charset="utf-8">
    <title>动画</title>
    <style type="text/css">
        .no1{
            width:100px;
            height: 100px;
            background-color: aliceblue;
        }
        .no1:hover{
            animation-duration: 1000ms;
            animation-delay: 200ms;
            animation-name: los23kgs;
            animation-iteration-count: 2;
            animation-direction: alternate;
        }
        @keyframes los23kgs {
            from{
                width: 150px;
                height: 150px;
                background-color: royalblue;
            }
            50%{
                width: 200px;
                height: 200px;
                background-color: crimson;
            }
            75%{
                width: 250px;
                height: 250px;
                background-color: sandybrown;
            }
            to{
                width: 300px;
                height: 300px;
                background-color: seagreen;
            }
        }
    </style>
</head>
<body>
    <p class="no1"></p>
    <br>
    <a href="../7.html">返回</a>
</body>
</html>
```
